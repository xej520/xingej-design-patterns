# xingej-design-patterns
目前测试用例包括：


第一、创建型设计模式


1、单例模式:
   A、饿汉式
   B、懒汉式
   C、双重检测方式(不建议使用)
   D、静态内部类方式
   E、枚举类方式
 
2、工厂模式：
   A、简单工厂模式(用的最多)
   B、工厂方法模式
   C、抽象工厂模式
   
3、建造者模式   

4、原型模式： A、浅复制；B、深复制

第二、结构性模式

1、适配器模式(记住：谁是调用方，被调用方，从而确定中间件/适配器；调用方有适配器的引用，适配器有被调用方的引用；调用方法调用适配器，适配器调用被调用者)

2、代理者模式：A、静态代理者；B、动态代理者
